## ğŸŒµ Getting Started

1. **Clone the repo:**
   ```sh
   git clone https://github.com/yourusername/plant-chat.git
   cd plant-chat
   ```
2. **Install dependencies:**
   ```sh
   flutter pub get
   ```
3. **Firebase setup:**
   - Add your `google-services.json` (Android) and `GoogleService-Info.plist` (iOS) to the project.

4. **Run the app:**
   ```sh

   ```
